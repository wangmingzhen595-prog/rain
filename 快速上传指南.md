# 🚀 快速上传到GitHub

## 方法1：网页直接上传（最简单，推荐）

### 步骤：

1. **打开GitHub仓库**
   - 访问：https://github.com/wangmingzhen595-prog/-
   - 如果仓库是空的，会看到 "Quick setup" 页面

2. **开始上传**
   - 点击 **"uploading an existing file"** 链接
   - 或者点击右上角 **"Add file"** -> **"Upload files"**

3. **选择文件**
   - 可以拖拽整个文件夹到页面
   - 或者点击 **"choose your files"** 选择文件
   - **建议按目录上传**（见下方文件清单）

4. **提交信息**
   ```
   初始提交：STM32雨滴传感器检测系统，已修复电压显示跳变问题
   ```

5. **提交**
   - 选择 **"Commit directly to the main branch"**
   - 点击 **"Commit changes"** 按钮

---

## 方法2：使用GitHub Desktop（功能更强大）

### 步骤：

1. **下载安装**
   - 访问：https://desktop.github.com/
   - 下载并安装 GitHub Desktop

2. **登录**
   - 打开 GitHub Desktop
   - 使用浏览器登录你的GitHub账号

3. **添加仓库**
   - 点击 **File** -> **Add Local Repository**
   - 点击 **Choose...** 选择项目目录：
     ```
     D:\stm32\stm32 project\12.2\8-2 DMA+AD多通道
     ```
   - 如果提示不是git仓库，勾选 **"Create a repository"**

4. **提交并推送**
   - 在左下角输入提交信息：
     ```
     初始提交：STM32雨滴传感器检测系统，已修复电压显示跳变问题
     ```
   - 点击 **"Commit to main"**
   - 点击 **"Publish repository"**
   - 选择远程仓库：`https://github.com/wangmingzhen595-prog/-.git`
   - 完成！

## 💡 上传技巧

1. **批量上传**：可以一次选择多个文件，支持拖拽
2. **按目录上传**：先上传 `User/`，再上传 `Hardware/`，依次进行
3. **检查文件**：上传后刷新页面，确认文件都在
4. **后续更新**：修改代码后，再次点击 "Upload files" 上传新文件
