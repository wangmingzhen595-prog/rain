========================================
快速上传步骤
========================================

方法1：网页直接上传（最简单）
----------------------------------------
1. 访问：https://github.com/wangmingzhen595-prog/-
2. 点击 "uploading an existing file" 或 "Add file" -> "Upload files"
3. 拖拽上述文件到页面（可以一次选择多个文件）
4. 在底部输入提交信息：
   初始提交：STM32雨滴传感器检测系统，已修复电压显示跳变问题
5. 点击 "Commit changes" 完成

方法2：使用GitHub Desktop（推荐）
----------------------------------------
1. 下载：https://desktop.github.com/
2. 安装后打开，登录GitHub账号
3. File -> Add Local Repository -> 选择项目目录
4. 如果提示不是git仓库，勾选 "Create a repository"
5. 点击 "Commit to main"
6. 点击 "Publish repository" -> 选择远程仓库
7. 完成上传
